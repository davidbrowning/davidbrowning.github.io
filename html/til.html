Today I learned

2/11/2016


Today I learned a lot about how to use vim. I am starting to practice, and improve my understanding of the language. Specifically, I learned how to jump up, down, left, and right using the hjkl keys. I learned that y is 'yank' and p is for 'paste'.


2/18/2016


Today I learned how to host a webpage using nodejs, a little about the express package, and several additional commands to use within vim. The specific vim commands are:

  1. o and O to insert a new line below and above the cursor respectively.
  2. 0 to jump to the beginning of a line
  3. :set number to display line numbers
  4. :noh to clear the highlighted text until the next search '/' command
  5. #G or #gg to jump to a specified line number


2/23/2016


Today I learned that you can download content directly from a page if you copy the URL and paste it into the downloads section of firefox. I also picked up another vim command, 'R' is for replace. It acts just like 'overtype' in ms word, namely, that it replaces all characters you type starting wherever you initiated the command.


9/1/2016


Today I learned that the reason you can't just boot in to safe mode with f8 is that the boot process is just too fast. Technically, the commands f8 or shift + f8 still work, but there isn't enough time for the operating system to recognize them, thus MS added in a way to reach troubleshooting if the OS failes to load 3(?) times.
I also picked up an incredibly useful powershell function:
[System.Data.SQL.SQLDataSourceEnumerator]::Instance.GetDataSources()
I found it on this article by Edwin Sarmiento. It queries all the SQL instances on your network, and returns the host and instance names of the SQL servers it finds.
